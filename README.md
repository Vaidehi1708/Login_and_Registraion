# Login_and_Registraion
This project is a console-based Login and Registration System developed  in C++. It allows new users to register by creating a unique username  and password, and enables registered users to securely log in by  validating their credentials against stored records. 

**Features**
- New user registration with username and password creation
- Duplicate username check to ensure unique user accounts
- Secure login validation against registered credentials
- Handling of incorrect login attempts with appropriate error messages
